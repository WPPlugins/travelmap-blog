Display an interactive TravelMap on your Wordpress blog. Quickly show and update your itinerary with the latest geolocation technology.

== Description ==

TravelMap plugin lets you embed an interactive travel map on your Wordpress blog fast and easy.  
Here is a **[DEMO](https://travelmap.net/blog/scandinavia-cycling-tours)** of what it looks like on a blog post.

You need a free TravelMap account to create and manage your Map.  
Don't worry, it only takes a few seconds to set up. Go to: [TravelMap.net](https://travelmap.net)

Features:

* Instantly show where you are, where you've been and where you're about to go
* Trace your itinerary using different colors for different transport modes
* One click geolocation: pin a new spot on your map to update your itinerary
* You can link photos, videos and articles to your Map

Note: This is the first version of the plugin.  
If you have any question or suggestion, don't hesite to [Contact us](https://travelmap.net/contact).

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/travelmap-blog` directory, or install the plugin through the WordPress plugins screen directly
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the TravelMap options page (via main menu) to configure the plugin

== Frequently Asked Questions ==

= Do I need to create an account? =
**Yes**, you need to create an account on [TravelMap.net](https://travelmap.net).

= Is it free? =
**Yes**, you can create your map and add as many spots as you like.
However, if you want to upload more than 100 photos on TravelMap to link them to your map, you need a [Premium account](https://travelmap.net/premium).

= How do I manage my itinerary? =
We plan to add a way for you to manage your itinerary directly from your WordPress admin but for now you need to do it on your TravelMap Admin.

You can find more information on the [Support page](https://travelmap.net/support) of TravelMap.
